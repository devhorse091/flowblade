## Flowblade

Data store and transformation library for data-driven applications.

### Examples

NextJs: https://flowblade-next-app.vercel.app/demo/duckdb

## Structure

```
.
├── examples
│   │── apps  
│   │   │── fastify-app
│   │   └── nextjs-app
│   └── shared  
│       │── db-sqlserver 
│       └── duckdb-openfoodfact
└── packages
    │── source-duckdb
    │── source-kysely
    │── sql-tag
    │── sql-tag-format
    └── (...)
```
